# arduino-rust-seed
`rust` project generator for arduino applications.

## dependencies

* python
* [cookiecutter](https://cookiecutter.readthedocs.io/)

## quickstart

### create arduino uno app

    cookiecutter git@github.com:ToddG/arduino-seed.git
